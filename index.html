Welcome to Sri Balaji Testing and Hallmarking Center

This is your complete management system. All data is stored on the server and shared across all devices on the same WiFi network.

Daily Attendance

  1. Click Attendance on home screen
  2. Allow camera and location permission when asked
  3. Wait for ✅ green location badge (must be at office)
  4. Face the camera clearly — photo is required
  5. Select your name from the dropdown
  6. Click CHECK-IN when arriving
  7. Click CHECK-OUT when leaving
  8. App returns to home automatically

Jeweller Request (F-29)

  1. Click JEWELLER → Submit Request
  2. Enter unique Invoice Number
  3. Select customer from dropdown
  4. Add jewellery items — weight auto-totals
  5. Sample count fills automatically (BIS rules)
  6. Click Submit & Generate F-29
  7. F-29 receipt prints automatically

F-30 Invoice

  1. Go to Admin → SBH Billing → F-30
  2. Type invoice number and press Enter
  3. F-29 data loads automatically
  4. Fill H/M quantity, rate for each item
  5. Enter weight returned & residue weight
  6. Click Generate F-30 Invoice

Billing Receipt

  1. Click BILLING on home screen
  2. Enter customer name and weight
  3. Fill optional fields (box no, delivery time)
  4. Click Generate & Print Receipt
  5. To edit: Admin → Billing Reports → Edit button

Logo Charges

  1. Go to Admin → SBH Billing → Customer Ledger
  2. Select customer
  3. Check Logo Charges option
  4. Price: ₹236 for ≤5 pieces, ₹45/pc beyond 5
  5. 18% GST applied automatically
  6. Click Print Logo Charges Invoice

Monthly Responsibilities

  • Check attendance records every month-end
  • Export all data backup (Admin → DB Info)
  • Review GST report for billing
  • Verify all F-29 have matching F-30
  • Update customer details if changed
  • Archive old records if needed

Important Tips

  • Always keep the server window open on the main PC — never close it
  • Invoice numbers must be unique — system will alert if duplicate
  • All devices must be on the same WiFi to share data
  • Take a backup weekly using Admin → DB Info → Export All Data
  • Camera on Android over WiFi needs HTTPS — use the GitHub Pages link for full camera access
  • Photo is required to check-in/out — face the camera clearly before clicking the button
  • Location must be at office — check-in/out only works within 150 m of Sri Balaji Center
  • Use the 🔔 bell icon to enable/disable duplicate invoice alert sounds

Sri Balaji Testing and Hallmarking Center — User Guide

For new admins and staff. Read this before operating the portal.

Daily Attendance

  1. Click ATTENDANCE on home screen
  2. Allow camera permission when prompted
  3. Select your name from dropdown
  4. Click CHECK-IN when arriving — app returns home automatically
  5. Later, click Attendance again and select CHECK-OUT
💡 Admin can view all attendance under Admin → Attendance Tab. Filter by daily/monthly/yearly.

Jeweller Request (F-29)

  1. Click JEWELLER on home screen
  2. Enter a unique Invoice Number — system alerts if duplicate or out of order
  3. Select customer from dropdown (details auto-fill)
  4. Add jewellery rows — enter description, pieces, and weight
  5. Total weight and sample count fill automatically (BIS rules)
  6. Press Enter/Tab to move between fields
  7. Click Submit & Generate F-29 to save and print
  8. First-time: F-22 Job Card + F-23 Sample Envelopes also print

F-30 Delivery Invoice

  1. Go to Admin → SBH Billing → F-30
  2. Type invoice number and press Enter — F-29 data loads automatically
  3. Enter H/M quantity, rate per item
  4. Enter weight of articles returned and residue weight
  5. GST (CGST + SGST) is auto-calculated
  6. Click Generate F-30 Invoice to print
⚠️ F-30 uses current date/time, not the F-29 date

Customer Billing Receipt

  1. Click BILLING on home screen
  2. Enter customer name and weight (required)
  3. Optional: items count, box number, delivery time
  4. Click Generate & Print Receipt
  5. To edit: Admin → Billing Reports → Edit button
  6. To reprint: Admin → Billing Reports → Reprint button

Logo Charges

  1. Go to Admin → SBH Billing → Customer Ledger
  2. Select the customer
  3. Tick the Logo Charges checkbox
  4. Rate: ₹5 per piece + 18% GST
  5. Enter piece count → total auto-calculates
  6. Click Print Logo Charges Invoice

Customer Management

  1. Go to Admin → SBH Billing → Customer Master
  2. Add new customers with GSTIN, CML, address, contact
  3. View ledger: all F-29 and F-30 transactions per customer
  4. Filter ledger by Daily / Monthly / Yearly

Monthly Responsibilities (Admin)

📊 Reports & Data
  • Export attendance report (CSV)
  • Check GST report → verify invoice order
  • Confirm all F-29 have matching F-30
  • Export DB backup from DB Info tab
🏢 Operations
  • Review non-conformity register (F-34)
  • Check equipment maintenance register (F-18)
  • Update staff training records (F-08)
  • Environmental conditions log (F-36)
⚙️ System Maintenance
  • Keep server PC always running
  • Weekly backup to USB drive
  • Verify all devices connect to same WiFi
  • Test alert tones are working

Important Tips for New Admins

  • Never close the server.js terminal window
  • Invoice numbers must be sequential — system will warn you
  • Use the 🔔 bell icon to toggle duplicate alert sound
  • All devices must be on same WiFi network
  • BIS logo is watermarked on all official documents
  • F-22 and F-23 print only on first creation of F-29
  • Edit billing receipts from Billing Reports tab
  • Take weekly backup — Admin → DB Info → Export
